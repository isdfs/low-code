Function deepMergeWithRules

  • deepMergeWithRules<T>(objects, rules): T

  • 深度合并对象，根据自定义规则处理冲突。

    Type Parameters

    • T

      对象的类型。

    Parameters

    • objects: T[]

      要合并的对象数组。

    • rules: Record<string, ((a: any, b: any) => any)>

      冲突处理规则映射表，键为对象属性，值为合并函数。

    Returns T

    返回合并后的新对象。

    Example

    const obj1 = { a: 1, b: { c: 2 } };
    const obj2 = { a: 2, b: { d: 3 } };
    const rules = {
      a: (x, y) => x + y,
      'b.c': (x, y) => Math.max(x, y)
    };
    const merged = deepMergeWithRules([obj1, obj2], rules);
    console.log(merged); // { a: 3, b: { c: 2, d: 3 } }
